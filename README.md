# Monitor de Movimento com Câmera Integrada à FPGA e Controle ARM

Projeto desenvolvido para a disciplina Projeto de Bloco: Sistemas Digitais Embarcados.

## TP1

Este repositório contém os primeiros artefatos técnicos do TP1:

- Planejamento inicial do projeto ARM-FPGA.
- Módulo combinacional em Verilog para classificação de movimento.
- Testbench funcional para validação da lógica.
- Evidências de síntese e simulação.
- Futuramente: rotina Assembly ARM64 executada no Raspberry Pi Zero 2 W.

## Estrutura

- `verilog_tp1/`: projeto Gowin e arquivos Verilog.
- `assembly_tp1/`: códigos Assembly ARM64.
- `docs/`: documentos e relatórios.
- `evidencias/`: prints e fotos do desenvolvimento.
